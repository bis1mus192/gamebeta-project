# GameBeta – сервис подписки на бета-тестирование игр

Современный одностраничный веб-интерфейс для каталога игр с возможностью бронирования бета-доступа, корзиной и локальным сохранением данных.

---

## Адрес рабочего проекта

Проект размещён на платформе **GitHub Pages** и доступен для просмотра по ссылке:  
[https://bis1mus192.github.io/gamebeta-project)  

## О проекте

**GameBeta** – это полностью клиентское веб-приложение, позволяющее пользователям:

- просматривать каталог игр, доступных для бета-тестирования;
- фильтровать игры по жанрам (RPG, Экшен, Симулятор, MMORPG);
- искать игры по названию;
- сортировать список по цене или по имени;
- добавлять выбранные игры в корзину с сохранением данных в `localStorage`;
- изменять количество ключей, удалять позиции;
- оформлять заказ с валидацией email.

Все данные хранятся локально в браузере, серверная часть отсутствует – проект создан в учебных целях для демонстрации навыков фронтенд-разработки.

## Функциональные возможности

### Для посетителя сайта

| Функция | Описание |
|---------|----------|
| **Главный экран** | Привлекательный баннер с призывом к действию |
| **Каталог игр** | Карточки игр с изображением, жанром, ценой и кнопкой добавления в корзину |
| **Фильтрация** | Быстрый выбор игр по жанру (кнопки «Все жанры», «RPG», «Экшен» и т.д.) |
| **Поиск** | Поиск по названию игры в реальном времени (с debounce) |
| **Сортировка** | По названию, по возрастанию/убыванию цены |
| **Корзина** | Просмотр добавленных игр, изменение количества, удаление, подсчёт итоговой суммы |
| **Оформление заказа** | Форма с валидацией email, имитация отправки заказа и очистка корзины |
| **Страница акций** | Информация о текущих скидках и бонусах |
| **Мои подписки** | Отображение текущего содержимого корзины (активные «подписки») |

## Технологический стек

| Технология | Назначение |
|------------|------------|
| **HTML5** | Семантическая разметка страниц |
| **CSS3** | Flexbox, Grid, медиа-запросы, адаптивный дизайн |
| **JavaScript (Vanilla JS)** | Вся клиентская логика: классы, работа с `localStorage`, фильтрация, сортировка, поиск, корзина |
| **Google Fonts** | Современные шрифты (по желанию можно добавить) |
| **Font Awesome** | Иконки (не используются, но можно добавить) |
| **Git** | Система контроля версий |
| **GitHub Pages** | Хостинг для публикации статического сайта |

## Структура проекта
```
project/

├── index.html          # главная страница (каталог игр)

├── init-main.js        # инициализация главной страницы
└── bin-logic.js        # логика страницы корзины
```
